### RGB 스펙트럼
```
RGB는 Red, Green, Blue 빛의 3원색을 뜻하는 줄임말입니다.
빛의 3원색인 빨강, 초록, 파랑이 섞이면 섞일수록 하얀색(빛)이 나오는 원리인데, 미술 시간에 배운 가산혼합이 바로 RGB랍니다!
쨍한 원색으로 표현되고 우리가 보는 모든 디스플레이 (ex 모니터, 스마트폰 등) 는 RGB가 적용됩니다.
웹에 올리는 모든 자료는 RGB로 선택하셔야 색상이 맑고 환하게 나온답니다!
```

### CMYK
```
CMYK는 Cyon, Magenta, Yellow, blacK의 줄임말입니다.
색의 3원색인 빨강, 파랑, 노랑을 뜻하며 마치 물감처럼 색을 섞을수록 탁해지는 감산혼합입니다.
같은 빨간색, 같은 파란색이어도 RGB 색상보다 탁하고 어두운 색을 띕니다.
우리가 보는 모든 인쇄물 (ex 프린터물, 잡지, 브로슈어 등) 이 모두 CMYK 작업물입니다.
```

 # 자바스크립트의 기본 규칙
 ```
- 하나의 실행문이 끝나면 마지막에 ;(세미콜론)을 붙여야 한다.

- 대소문자를 구분해야 한다.
  (ex) clickHandler > clickHandler , clickhandler 

- 가독성을 위해 들여쓰기 한다.

- 주석문은 한줄 주석과 여러줄 주석이 있다
  // > 한줄 , /**/ > 여러줄 주석

 ```
